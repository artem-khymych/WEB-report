В CSS існує кілька різних типів селекторів, які дозволяють вибирати елементи HTML для застосування стилів. Наприклад:

  1. Елементний селектор: Вибирає всі елементи з вказаним ім'ям тега. Приклад:

                    
                    p{
                        color: blue;
                    }

    Цей код застосовує стиль до всіх <p> елементів, роблячи текст у них синім.

  2. Ідентифікаторний селектор: Вибирає елемент з конкретним id атрибутом. Приклад:

                    
                    #header {
                        background-color: lightgray;
                    }

    Це застосовує стиль до елементу з id="header", змінюючи його фон на світло-сірий.

  3. Класовий селектор: Вибирає всі елементи з певним значенням класу. Приклад:

                    
                   .highlight {
                        font-weight: bold;
                   }

    Цей селектор стилізує всі елементи з класом highlight, роблячи їх текст жирним.

  4. Груповий селектор: Дозволяє вибрати кілька селекторів одночасно та застосувати до них одні й ті ж стилі. Приклад:

                    
                  h1, h2, h3 {
                        color: red;
                  }

    Цей селектор стилізує всі елементи заголовків першого, другого та третього рівня, роблячи їх текст червоним

  5. Дочірній селектор: Вибирає елементи, які є безпосередніми нащадками іншого елемента. Приклад:

                    
                  div > p {
                        margin-left: 20px;
                  }

    Цей селектор стилізує всі <p> елементи, які є безпосередніми дочірніми елементами <div>, додаючи лівий відступ 20px.
