Існують кілька способів підключення стилів CSS до HTML-сторінки, а саме:

  1. Вбудований стиль: Використовується тег <style> для вказання CSS-стилів прямо в HTML-файлі. Приклад: 

                        
                        <!DOCTYPE html>
                        <html>
                        <head>
                            <title>Приклад вбудованого стилю</title>
                            <style>
                                body {
                                    font-family: Arial, sans-serif;
                                }
                                h1 {
                                    color: blue;
                                }
                            </style>
                        </head>
                        <body>
                            <h1>Привіт, світ!</h1>
                            <p>Це приклад вбудованого стилю.</p>
                        </body>
                        </html>

  2. Зовнішній файл: Створення окремого CSS-файлу і підключення його до HTML-сторінки за допомогою тегу <link>. Приклад: 

                        
                        <!DOCTYPE html>
                        <html>
                        <head>
                            <title>Приклад зовнішнього CSS</title>
                            <link rel="stylesheet" href="styles.css">
                        </head>
                        <body>
                            <h1>Привіт, світ!</h1>
                            <p>Це приклад зовнішнього CSS.</p>
                        </body>
                        </html>

  3. Inline-стиль: Використовується атрибут style для вказання CSS-стилів безпосередньо в тегах HTML. Приклад: 

                        
                        <!DOCTYPE html>
                        <html>
                        <head>
                            <title>Приклад inline-стилю</title>
                        </head>
                        <body>
                            <h1 style="color: blue;">Привіт, світ!</h1>
                            <p style="font-family: Arial, sans-serif;">Це приклад inline-стилю.</p>
                        </body>
                        </html>
